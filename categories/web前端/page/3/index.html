еҺ»зҷҫеәҰйқўиҜ•еҹәзЎҖжһ¶жһ„йғЁзҡ„еүҚз«ҜејҖеҸ‘е®һд№ з”ҹпјҢж„ҹи§үеә”иҜҘи·Әеҫ—еҫҲжғЁгҖӮзҷҫеәҰдёҠжө·з ”еҸ‘дёӯеҝғжҢәеҒҸзҡ„пјҢзҰ»еј жұҹиҝҳжҢәиҝңзҡ„пјҢиҝҳиҰҒжү“иҪҰиҝҮеҺ»гҖӮеҲ°дәҶзҷҫеәҰйӮЈиҫ№й—ЁеҚ«з®Ўзҡ„жҢәдёҘзҡ„пјҢиҝҳиҰҒйқўиҜ•е®ҳжқҘжҺҘгҖӮеҗҺжқҘпјҢе°ұдёҖдёӘзЁӢеәҸе‘ҳжқҘжҺҘжҲ‘пјҢд№ҹе°ұжҳҜжҲ‘зҡ„йқўиҜ•е®ҳгҖӮеҗҺжқҘе°ұеңЁд»–д»¬дёүжҘјдёҖдёӘдј‘жҒҜзҡ„ең°ж–№ејҖе§ӢйқўиҜ•гҖӮйҰ–е…Ҳе°ұжҳҜиҮӘжҲ‘д»Ӣз»ҚдәҶпјҢз„¶еҗҺе°ұжҳҜblablaжҲ‘дёҖдәӣз»ҸеҺҶйЎ№зӣ®иҖҢе·ІдәҶгҖӮд»–ејәи°ғдәҶдәҶдёҖдёӢеүҚз«Ҝзӣёе…ізҡ„йЎ№зӣ®з»ҸеҺҶпјҢдҪҶжҳҜи®ІйҒ“зҗҶжҲ‘жҳҜзңҹеҝғжІЎжңүеүҚз«Ҝзҡ„йЎ№зӣ®з»ҸеҺҶпјҢжүҖд»Ҙд№ҹиҜҙдёҚдёҠд»Җд№ҲгҖӮе”ҜдёҖеҒҡзҡ„д№ҹе°ұжҳҜжҲ‘иҮӘе·ұзҡ„дёӘдәәдё»йЎөпјҢиҝҳеҹәжң¬дёҠйғҪжҳҜйқҷжҖҒйЎөйқўгҖӮз„¶еҗҺејҖе§ӢжӯЈејҸйқўиҜ•пјҢд»–йҰ–е…Ҳй—®дәҶжҲ‘ж Үзӯҫе’Ңж Үзӯҫжңүд»Җд№ҲеҢәеҲ«пјҢз„¶еҗҺжҲ‘иҜҙдәҶе’Ңзҡ„жң¬иҙЁеҢәеҲ«жҳҜд»–д»¬зҡ„й»ҳи®Өdisplay еұһжҖ§дёҚеҗҢпјҢз„¶еҗҺд»–еҸҲй—®дәҶжңүжІЎжңүе…¶д»–зҡ„дёҚеҗҢпјҢжҲ‘е№¶дёҚзҹҘйҒ“гҖӮз„¶еҗҺд»–жҺҘзқҖй—®дәҶеёёи§Ғзҡ„еқ—зә§ж Үзӯҫе’ҢиЎҢеҶ…ж Үзӯҫжңүе“ӘдёҖдәӣгҖӮжҺҘзқҖд»–й—®дәҶжҲ‘жё…йҷӨжө®еҠЁзҡ„зҹҘиҜҶпјҢжҲ‘еҸӘжҳҜдёҙж—¶и®°дҪҸдәҶдёҖдәӣжё…жҘҡжө®еҠЁзҡ„ж–№жі•пјҢдҪҶжҳҜиҝҳжҳҜжІЎжңүеј„жҮӮе®һйҷ…зҡ„еҺҹзҗҶпјҢжүҖд»Ҙеңәйқўд№ҹжҳҜжҜ”иҫғзҡ„е°ҙе°¬гҖӮд»–й—®дәҶжҖҺд№ҲдјҳеҢ–зҪ‘йЎөпјҢиҜҙеҸҜд»ҘжҖҺд№ҲеҒҡгҖӮжҲ‘дёҚзҰҒдёҖе–ңпјҢжҠҠд№ӢеүҚеҮҶеӨҮзҡ„дёңиҘҝиғҢеҮәжқҘпјҢеҲҡиҜҙеҲ°з¬¬дёҖжқЎпјҢеҮҸе°‘ httpиҜ·жұӮпјҢд»–е°ұй—®жҲ‘еҰӮдҪ•еҮҸе°‘ httpиҜ·жұӮгҖӮе°ҙе°¬пјҢдёҖдёӢеӯҗggпјҢжҲ‘д№ҹжғідёҚиө·жқҘеҰӮдҪ•еҮҸе°‘httpиҜ·жұӮгҖӮеҗҺжқҘд»–зңӢжҲ‘иҜҙдёҚдёҠжқҘе°ұи®©жҲ‘з»§з»ӯеҫҖдёӢиҜҙпјҢжҲ‘еҸҲиҜҙдәҶеҺӢзј©пјҢд»ҘеҸҠд»Јз ҒзІҫз®Җд№Ӣзұ»зҡ„гҖӮз„¶еҗҺд»–еҸҲй—®дәҶе…·дҪ“зҡ„пјҢжҲ‘иҜҙдәҶдёҖдёӢпјҢд»–еҘҪеғҸд№ҹдёҚжҳҜзү№еҲ«ж»Ўж„ҸгҖӮжҲ‘зҡ„з®ҖеҺҶпјҢе°јзҺӣпјҢжҲ‘дёәд»Җд№ҲиҰҒеҶҷдёӘеҜ№дәҺhtml5жңүе……еҲҶзҡ„и®ӨиҜҶпјҒпјҒпјҒжҲ‘еӨ©еӨ©еҸҲжҮ’еҫ—ж”№пјҢжҜҸдёӘйқўиҜ•е®ҳйғҪиҰҒй—®жҲ‘иҝҷдёӘй—®йўҳгҖӮдёҚиҝҮжҲ‘дәӢе…Ҳд№ҹеҮҶеӨҮдәҶпјҢе°ұжҠҠдёҖдәӣеҹәзЎҖзҡ„зү№жҖ§иҜҙдәҶдёҖдёӢгҖӮд»–иҜҙиҝҳжңүеҲ«зҡ„пјҢз„¶еҗҺжҸҗйҶ’жҲ‘зј“еӯҳд№Ӣзұ»зҡ„пјҢз„¶еҗҺе°ұиҜҙиө·жқҘsessionStorageпјҢlocalStroageд№Ӣзұ»зҡ„пјҢе№¶й—®дәҶжҲ‘е’Ңcacheд№Ӣй—ҙзҡ„еҢәеҲ«д№Ӣзұ»зҡ„гҖӮиҝҷйҮҢйқўжңүдёӘй—®йўҳд»–зӮёдәҶжҲ‘дёҖдёӢпјҢд»–иҜҙдҪ зЎ®е®ҡlocalStorageеңЁжң¬ең°жҳҜдёҚдјҡеҲ йҷӨзҡ„д№ҲпјҢеҫҲжҳҺжҳҫжҳҜдёҚдјҡзҡ„гҖӮз„¶еҗҺд»–й—®еҜ№дәҺзј“еӯҳиҜ·жұӮжңүжІЎжңүдәҶи§ЈпјҢеҫҲжҳҺжҳҫжҲ‘е№¶жІЎжңүд»Җд№ҲдәҶи§ЈгҖӮд»–еҸҲй—®дәҶжҲ‘дёҖдәӣеҜ№дәҺе“ӘдәӣејҖжәҗзҡ„жЎҶжһ¶жҜ”иҫғзҶҹжӮүпјҢйӮЈд№ҹе°ұеҸӘжңүjQueryе’ҢbootstrapгҖӮд»–й—®жҲ‘еҜ№дәҺиҝҷдәӣејҖжәҗжЎҶжһ¶зҡ„жӣҙж–°жё…дёҚжё…жҘҡпјҢеҫҲжҳҺжҳҫпјҢе®қе®қдёҚжё…жҘҡе•ҠпјҒпјҒпјҒз„¶еҗҺд»–й—®дәҶдёҖзі»еҲ—еӣҫж Үж”№еҸҳйўңиүІеҰӮдҪ•е®һзҺ°пјҢе®һзҺ°йј ж ҮжҢӘдёҠеҺ»пјҢе°ұж”№еҸҳйўңиүІгҖӮиҝҷдёӘжҲ‘иҜҙз”Ёhoverж”№еҸҳcss еұһжҖ§пјҢдҪҶд»–еә”иҜҘжҳҜиҙЁз–‘иҝҷж ·зҡ„еҒҡжі•еҗ§гҖӮеҗҺжқҘд»–еҸҲиҜҙйҒ“bootstrapе…¶е®һжңүиҝҷж ·зҡ„е®һзҺ°пјҢе…¶е®һжҲ‘еҘҪеғҸд№ҹзңӢеҲ°иҝҮпјҢдҪҶжҳҜжҲ‘д№ҹжІЎе…іжіЁиҝҮжҳҜеҰӮдҪ•е®һзҺ°зҡ„пјҢзңҹжҳҜжӮІеү§гҖӮжҺҘзқҖд»–й—®дәҶjsзҡ„й—®йўҳгҖӮйҰ–е…Ҳй—®дәҶдёҖдёӢдәӢд»¶жөҒпјҢиҝҷдёӘең°ж–№жҲ‘иҜҙй”ҷдәҶпјҢдәӢд»¶жөҒеә”иҜҘжҳҜиЎҘиҙ§пјҢеӨ„зҗҶпјҢеҶ’жіЎгҖӮжҲ‘жҠҠйЎәеәҸиҜҙеҸҚдәҶпјҢеҸҚжӯЈдәӢд»¶жөҒиҝҳй—®дәҶиҜҰз»Ҷзҡ„й—®йўҳпјҢдҪҶжҳҜжҲ‘еҜ№иҝҷдёӘе№¶дёҚжҳҜзү№еҲ«дәҶи§ЈпјҢжүҖд»ҘиҜҙзҡ„д№ҹжҳҜдёҚеҘҪзҡ„гҖӮеҗҺжқҘеҮәдәҶдёҖдёӘjsзҡ„йўҳзӣ®пјҢеҰӮдҪ•е®һзҺ°еғҸtrimдёҖж ·еҺ»жҺүдёӨз«Ҝзҡ„з©әзҷҪпјҢжҲ‘з”ЁjsеҶҷдәҶдёҖдёӢпјҢйҮҢйқўжңүеҫҲеӨҡй—®йўҳпјҢе…·дҪ“е°ұжҳҜдёҚиҜҙдәҶгҖӮеҗҺжқҘд»–жҸҗзӨәиҜҙжҳҜз”ЁжӯЈеҲҷиЎЁиҫҫејҸпјҢжҲ‘иҜҙжҲ‘еҸӘдјҡpythonзҡ„пјҢз„¶еҗҺд»–иҜҙеҸҜд»ҘгҖӮдҪҶжҳҜе®қе®қд№ҹеҶҷдёҚеҮәжқҘе•ҠпјҢз„¶еҗҺд»–иҜҙйӮЈдҪ е°ұиҜҙдёӢеҺҹзҗҶеҗ§гҖӮз„¶иҖҢжҲ‘е№¶дёҚзҹҘйҒ“еҰӮдҪ•еҒҡпјҢз„¶еҗҺе°ұзҪўдәҶгҖӮ з¬¬дәҢдёӘйқўиҜ•е®ҳе°ұй•ҝеҫ—жӣҙеғҸзЁӢеәҸе‘ҳдәҶпјҢзӣҙжҺҘз©ҝдёӘжӢ–йһӢе°ұиҝҮжқҘдәҶгҖӮд»–жҳҜзӣҙжҺҘи®©жҲ‘еҶҷд»Јз Ғзҡ„пјҢз¬¬дёҖйўҳеҰӮдҪ•е®һзҺ°дёүеҲ—еёғеұҖпјҢдёӨиҫ№еҗ„100pxпјҢдёӯй—ҙиҮӘеҠЁжӢ“еұ•пјҢжҲ‘иҜҙдәҶз”ЁfloatжҲ–иҖ…css3зҡ„ж–°еұһжҖ§гҖӮд»–зӣҙжҺҘи®©жҲ‘еҶҷд»Јз ҒдәҶпјҢз„¶еҗҺе°ұеҶҷдёҖеҶҷпјҢдј°и®Ўд№ҹдёҚеӨӘеҜ№гҖӮз¬¬дәҢдёӘпјҢд»–иҜҙieе’Ңchromeзӯүз»‘е®ҡзҡ„дәӢд»¶зҡ„ж–№жі•дёҚеҗҢпјҢеҰӮдҪ•еҶҷдёҖдёӘеҮҪж•°иҝӣиЎҢдёҚеҗҢзҡ„еӨ„зҗҶгҖӮиҝҷдёҖйўҳжҲ‘е№¶дёҚзҹҘйҒ“еҰӮдҪ•еҢәеҲҶпјҢжүҖд»Ҙд№ҹе°ұжІЎеҶҷгҖӮз¬¬дёүйўҳпјҢд»–иҜҙжңүжІЎжңүеҒҡиҝҮз§»еҠЁз«Ҝзҡ„ејҖеҸ‘пјҢжҲ‘иҜҙжІЎжңүгҖӮз„¶еҗҺжҲ‘е°ұиҖҒе®һиҜҙдәҶпјҢе…¶е®һиҮӘе·ұе№¶жІЎжңүжҺҘи§ҰеӨӘеӨҡзҡ„еүҚз«ҜгҖӮз„¶еҗҺеҘ№иҜҙд№ҹзңӢдәҶжҲ‘з®ҖеҺҶжІЎд»Җд№ҲеүҚз«Ҝзҡ„з»ҸеҺҶд№Ӣзұ»зҡ„пјҢз„¶еҗҺе°ұжҳҜиҜҙиҜҙпјҢз„¶еҗҺе°ұжІЎдәҶгҖӮ жҖ»з»“дёҖдёӢпјҢиҝҷж¬ЎйқўиҜ•з»ҸеҺҶи®ІйҒ“зҗҶе№¶дёҚжҳҜзү№еҲ«еӣ°йҡҫзҡ„гҖӮеүҚз«Ҝзҡ„д№ҰжҲ‘д№ҹжҳҜзңӢдәҶеҘҪеҮ жң¬пјҢдҪҶе…¶е®һе№¶жІЎжңүд»Җд№Ҳзү№еҲ«зҡ„ж„ҸжҖқпјҢиҝҳжҳҜеҒҡйЎ№зӣ®жҲҗй•ҝзҡ„жҜ”иҫғеҝ«гҖӮжүҖд»ҘпјҢиө¶зҙ§е®һд№ пјҢе®һйҷ…еҸӮдёҺйЎ№зӣ®еҜ№дәҺжҲ‘жқҘиҜҙе®һеңЁжҳҜеӨӘйҮҚиҰҒдәҶгҖӮ
